### <h2> About

Project created in order to learn web development with Flask and Vue.js.

#### Application includes features such as : 

* Registration
* Login 
* Global wall for posting notes
* GUI for adapting weights of the artificial neuron 
* Test coverage 

#### Technologies used : 
* flask
* flask-login 
* flask-marshmallow
* flask-migrate
* unittest
* flask-testing
* vue.js 
* vue-resource 
* vue-router 
* css3  
* html5
